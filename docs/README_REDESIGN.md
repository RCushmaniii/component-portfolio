## üêõ Troubleshooting

### Theme not changing?
- Check that ThemeContext is wrapping your app
- Verify theme state is being passed correctly
- Clear browser cache

### Animations not smooth?
- Ensure Tailwind config is loaded
- Check for CSS conflicts
- Test in different browsers

### Mobile menu not working?
- Verify useState is imported
- Check click handlers
- Test on actual mobile device

### Colors look wrong?
- Verify theme variable is correct
- Check component receives theme prop
- Review CSS specificity
